# TIC / TAC / TOE — Neon Grid Protocol v2.0

A futuristic, neon-cyberpunk themed Tic-Tac-Toe web application built using Lovable. Play locally, challenge an AI protocol, or jump into real-time online matchmaking.

## 🎮 Game Modes

Choose your protocol. All online matches sync in real time:

1. **Local 1v1:** Two players, one device. Classic pass-and-play to challenge a friend right next to you.
2. **Vs Bot:** Test your skills against an AI system featuring three distinct difficulty levels:
   * **Easy**
   * **Intermediate**
   * **Impossible**
3. **Random Match:** Auto-match with an online rival. Check for available players and instantly jump into a game without needing codes.
4. **Private Room:** Host or join with a code. Create a private match and share the room code to duel your friends online.

## 🎨 Design & Tech

* **Visual Theme:** Cyan and magenta high-contrast Neon Cyberpunk grid aesthetic.
* **Built With:** Developed using modern rapid prototyping on Lovable.
